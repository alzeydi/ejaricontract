DEWA Premises Number — what it is
and where to find it

Updated April 2026 · Reference guide for Dubai tenants and landlords

The DEWA Premises Number is a 9-digit code that uniquely identifies every property in Dubai. You need it to activate electricity and water through the DEWA Move In service, and it appears on several official documents. This page explains what the number is, where to find it, how it differs from other Dubai property identifiers, and what to do if you cannot locate it.

i

What is the DEWA Premises Number?

It is a 9-digit numeric identifier assigned by Dubai Electricity and Water Authority (DEWA) to every metered property in Dubai — apartments, villas, commercial units, and warehouses. The number identifies the physical unit, not the customer. It does not change when tenants come and go, and it is the same whether you have an active DEWA account or not.
Format: 9 digits, no letters, no spaces. Example: 123456789. Sometimes printed with leading zeros.
i

Why you need the Premises Number

DEWA uses the Premises Number to look up the property in its system before opening a new account. You will be asked for it when you:
  • Activate DEWA as a new tenant (Move In application)
  • Transfer an existing DEWA account between properties
  • Register or renew Ejari (the field is required on the contract form)
  • Report a fault, request a meter reading, or query a bill
i

Where to find the DEWA Premises Number

The number appears on at least one of these documents — usually all of them.
Ejari certificateLabelled "Premises No." or "DEWA Premise No." in the property section
Title DeedOn the property details page (printed or sold copy)
Previous DEWA billTop of any bill issued for the unit (if a previous tenant had service)
DEWA mobile appUnder My Properties if you've ever held an account at the address
Building managementReception or facilities team usually has a list for the whole building
💡 If you don't have any of the above yet, ask the landlord or property agent — they should be able to provide it from their records before you sign the tenancy contract.
i

How it differs from other Dubai property numbers

Several Dubai property identifiers look similar at first glance. They are not interchangeable.
DEWA Premises Number9 digits. Identifies the property. Stays the same forever.
DEWA Account NumberVaries. Identifies a customer billing account. Changes with each new tenant.
Ejari registration numberFormat like 22-99999/2026. Identifies the tenancy contract, not the property.
Makani number10 digits. Government-wide geolocation code for the building entrance, used by emergency services.
Plot numberIdentifies the land plot in DLD records. Multiple units share one plot.

⚠️

Common mistakes when using the Premises Number

  • Confusing it with the DEWA Account Number — the previous tenant's account number won't open service for you
  • Dropping leading zeros — if the number starts with a 0, you must include it (DEWA stores 9 digits, not "as printed")
  • Using a Premises Number from a different unit — common in older buildings where multiple flats share similar numbering. Cross-check against the Ejari and Title Deed
  • Outdated number after building renumbering — rare, but DEWA occasionally renumbers buildings. If your old number is rejected, ask the building or DEWA for the current one
